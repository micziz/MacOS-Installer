# MacOS Installer

Bash installer to easaly install packages on macos

TLDR

```
git clone https://github.com/micziz/MacOS-Installer && cd MacOS-Installer
```

Clone the repo:
```
git clone https://github.com/micziz/MacOS-Installer
```
then cd into it
```
cd MacOS-Installer
```
and run the following command:
```
sh mainintel.sh
```


(for m1: 
```
sh mainm1.sh
```
)
